# Security Policy

## Data Handling

All AI processing in our workshops happens locally on your machine. No data is transmitted to external servers unless explicitly required for a specific exercise.

## Local Processing Benefits

- **Privacy**: Your data never leaves your device
- **Security**: No API keys or credentials are exposed
- **Compliance**: Suitable for sensitive data handling
- **Offline**: Works without internet connection

## Best Practices

1. Always verify AI outputs before using them in production
2. Keep your local models updated
3. Don't share sensitive information in workshop chat interfaces
4. Clean up containers and volumes after workshops

## Container Security

- Use official images only
- Run containers with minimal privileges
- Keep Docker Desktop updated
- Review container networking settings

## Reporting Issues

If you discover a security vulnerability during our workshops, please report it to the facilitator immediately.

## Questions?

Contact the workshop organizer if you have questions about security practices.
